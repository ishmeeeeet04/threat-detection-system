# 🛡️ AI-Powered Threat Detection & Log Analysis System

A machine learning-based cybersecurity tool that analyzes system and network logs
to detect anomalies, identify threats, and display real-time alerts on an
interactive dashboard.

## 🚀 Features
- Parses system authentication logs
- Detects failed login attempts, unusual IPs, and suspicious activity
- Uses Isolation Forest ML model for anomaly detection
- Generates threat scores (0-100) for each event
- Interactive dashboard with real-time charts and alerts

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **Machine Learning:** Scikit-learn (Isolation Forest)
- **Database:** SQLite
- **Frontend:** HTML, CSS, JavaScript, Chart.js
